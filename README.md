\# Bestell-App 🍕



A food ordering app inspired by Lieferando, built with HTML, CSS and JavaScript.



\## Features



\- Restaurant info with hero image

\- Dynamic menu rendering from database

\- Shopping cart with add/remove items

\- Order total calculation

\- Responsive design with mobile cart dialog

\- Sticky desktop cart



\## Technologies



\- HTML

\- CSS

\- JavaScript
